using tvm::internal::enable_for_t = typedef std::enable_if_t<(... || (std::is_same_v<T, Base> || derives_from<T, Base>())), int>

Used to enable a function for a list of types.

To have a function work for T equal or deriving from any B1, B2, ... or Bk where Bi are types. Use as template<typename T, enable_for_t<T,B1, B2, ..., ..., Bk>=0>

◆ derives_from()

template<typename T , template< typename... > class Base>
constexpr bool tvm::internal::derives_from ( )

Check if class T derives from the templated class Base.

Check if class T derives from the non-templated class Base This returns false if Base is not a class.

This relies on tvm::internal::is_base::check: if T derives from Base, the overload returning std::true_type will be selected, otherwise, it will be the one returning std::false_type.
