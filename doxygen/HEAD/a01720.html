Detailed Description

template<typename Derived>
struct tvm::utils::internal::RefSelector< Derived >

Shortcut to an internal Eigen type to store expressions or matrices.

When keeping internally a reference to an Eigen object, we need to have different behaviors depending on wether the object has a large memory and should not be copied or is a lightweight proxy. For matrix, we need to keep a const ref, while for matrix expression we need to take a copy of the expression.This is exactly the purpose of ref_selector, that is used to this effect in e.g.CWiseBinaryOp.
