Detailed Description

Representation of a Variable, i.e.

a point in a space.

A variable can only be constructed from a Space, in which case we say it is a base primitive, or from an other variable, by derivation with the dot() operator.

◆ isEuclidean()

bool tvm::Variable::isEuclidean ( ) const

Check if the variable lives in a Euclidean space.

Pay attention that for a variable v, v.isEuclidean() is not necessarily equal to v.space().isEuclidean(): if v is a derivative of a variable linked to a non-Euclidean space, it lives in the tangent space which is Euclidean.
